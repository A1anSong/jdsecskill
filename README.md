## 目录结构

***/hook:*** 使用frida hook数据的保留脚本

***convert.py:*** 将抓包数据转换为请求数据的脚本（使用Fiddler保存的数据）

***jd_user.py:*** 封装的JDUser类

***main.py:*** 主程序

***uers.py.example:*** 需要填写抓包获取的一些用户信息，其中是必填项

***页面请求.md*** 记录一些页面请求的参数
